# tg-bot-service

create .env file based on .env_example to run this bot

## Команды бота

### Админские (только в личке, только `ADMIN_USER_ID`)

| Команда | Описание |
| --- | --- |
| `/change_tags @u1,@u2,...` | Задать дефолтный список тегов (используется как fallback при упоминании MR, если у сервиса нет назначенных мейнтейнеров). Пустой аргумент очищает список. |
| `/set_maintainers <service> @u1,@u2,...` | Привязать список мейнтейнеров к имени сервиса. Имя сервиса — это сегмент пути в URL до `/-/merge_requests/`. Пустой список — удаляет привязку. |
| `/del_maintainers <service>` | Удалить мейнтейнеров для сервиса. |
| `/list_maintainers` | Показать текущий маппинг сервис → мейнтейнеры. |

Маппинг мейнтейнеров переживает перезапуски (хранится в `data/maintainers.json`). `change_tags` пока живёт только в памяти.

### Реакции на сообщения (любой чат)

Текст нормализуется (нижний регистр, без пробелов и пунктуации), затем матчится сначала по точному совпадению, потом по подстроке.

**Точное совпадение:**

| Триггер | Ответ |
| --- | --- |
| `господи забери меня отсюда` | «Хлопчик, остуди свою жепу, услышал тебя родной» |
| `видишь french bakery` | «А как на счет FUCK PARENTS?» |
| `продупал` | «Я пока не у компа, не смогу быстро посмотреть» |
| `тест команда упала` | «Я не у компа сейчас, не смогу оперативно посмотреть» |
| `накидайте` | «Накидайте за обе щеки бедолаге, благословлю» |

**Подстрока (без нормализации):**

| Триггер | Ответ |
| --- | --- |
| `обед` (с пробелом по краям) | «Приятного аппетита» |
| `бургер` | «господи закажи меня прямо сейчас» |
| URL с `GIT_MR_URL` | «Накидайте аппрувов» + блок тегов (см. ниже). Теги добавляются только в рабочее время MSK (Пн–Пт, 9–18). |

**Логика тегов на MR:**

- ≥2 мейнтейнеров у сервиса → `Мейнтейнеры: @u1 @u2 ...`
- 1 мейнтейнер → `Мейнтейнеры: @maintainer @random` (рандом тянется из `change_tags` и не пересекается с мейнтейнером)
- Сервис не зарегистрирован / нет мейнтейнеров → 2 случайных из `change_tags` без лейбла
| `дейли` / `дэйли` | Ссылка из `DAILY_LINK` |
| `ДИТ` | Анимация `assets/god.mp4` с цитатой «ДИТ» |
